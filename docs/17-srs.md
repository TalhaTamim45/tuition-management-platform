# Tuition Management Platform - Software Requirements Specification (SRS)

## 1. Introduction
This Software Requirements Specification (SRS) document details the functional and non-functional requirements for the Tuition Management Platform MVP.

## 2. Overall Description
The system is a multi-role web platform. Users can join as tutors (offering teaching services) or guardians (seeking tutors). The platform acts as a matchmaker without involving direct transactions or live chatting.

## 3. System Features
- **User Authentication:** Sign up/in, password security, session persistence.
- **Tutor Profile:** Searchable resume detailing subjects, location preferences, and expected pay.
- **Tuition Post Board:** Public listings of open tutoring positions.
- **Direct Application:** Workflow enabling tutors to express interest and parents to hire them.
- **Admin Control:** Administrative interface to ensure safety and moderate listings.

## 4. External Interface Requirements
- **User Interface:** HTML5, CSS3, responsive layouts.
- **Software Interface:** Node.js/Express REST APIs, relational database.
