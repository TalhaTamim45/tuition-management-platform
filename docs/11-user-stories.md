# Tuition Management Platform - User Stories

| Story ID | User Role | Description | Story Points |
|---|---|---|---|
| US-01 | User | As a new user, I want to register with my role (Tutor/Guardian) so that I can access customized features. | 3 |
| US-02 | Tutor | As a tutor, I want to complete my profile (education, experience, subjects) so that guardians see my qualifications. | 5 |
| US-03 | Guardian | As a guardian, I want to create a tuition post with details (class, salary, subjects, location) so tutors can apply. | 3 |
| US-04 | Tutor | As a tutor, I want to browse and search tuition jobs so that I can find jobs that match my criteria. | 5 |
| US-05 | Tutor | As a tutor, I want to apply for a tuition job with an optional message so the guardian knows my interest. | 3 |
| US-06 | Guardian | As a guardian, I want to view applicants for my post and accept or reject them so I can finalize a tutor. | 5 |
| US-07 | Admin | As an admin, I want to block users and delete inappropriate posts so the platform remains safe. | 5 |
